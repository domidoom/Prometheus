What it does

One button between you and your Warden

Press the on-screen orb or hit F9 from anywhere. A beep marks the start; you talk; another beep marks the end. The hologram pulses while it thinks, then speaks the reply back. Press again at any time to interrupt.

  • Push-to-talk via on-screen button or global F9 hotkey
  • Replies stream sentence-by-sentence — no waiting for the full answer
  • Press again to barge in and cut off the reply
  • Same Warden agent that answers Slack, WhatsApp, email, web

How it works

Local ears and mouth. Server-side brain.

Whisper transcribes your voice on the device. The transcript — and only the transcript — goes to your Warden over the same authenticated session you already use. The reply streams back as text. Kokoro speaks it locally.

  • No new account — uses your existing email-verified Warden session
  • Audio never leaves your machine
  • Reply text streams over SSE; speaking starts before the agent finishes
  • Works fully offline for STT/TTS once models are cached

Ways it simplifies further

Skip the keyboard for the things that don't need one

Half the requests we send to chat assistants don't need a typed message. They need a sentence — while you're walking, cooking, holding something, or just thinking out loud. Warden Voice closes that gap by reusing the agent and tools you already configured.

  • Dictate emails, tasks, and notes hands-free
  • Ask follow-up questions about a document without opening it
  • Trigger any automation by speaking the same phrasing you'd type
  • Drops onto a tiny side monitor as a dedicated assistant surface

Privacy

Your voice stays on the device

Local STT

Whisper runs on your CPU or GPU. The audio is decoded, transcribed, and discarded — it never leaves the machine.

Text only over the wire

Only the transcript and the reply text traverse the network — over the same authenticated session your Warden already uses.

No new credentials

Uses the email-verified Warden session you already created — no extra auth, no API keys, no cloud accounts.

Install

Three clicks to your first conversation

Download the installer, sign in, and talk. No terminal, no dependencies, no configuration files to edit by hand.

  • Single .exe installer — downloads everything it needs
  • Configuration wizard opens automatically after install
  • Runs as a normal desktop app with a system-tray icon
1

Run the installer

Download and run WardenSetup.exe. It installs Warden and the speech models in one pass.

2

Sign in

The configuration window opens automatically. Enter your Warden URL and log in with your existing account — same credentials as the web app.

3

Pick your group

Choose which Warden group your voice turns land in. You can change this later from the settings.

4

Talk

Warden launches. Press the hologram or hit F9 from anywhere — you're live.
